# Post-MVP Roadmap

Features and enhancements to add after the MVP is complete and stable.

---

## User Accounts & Auth
- [ ] User registration and login (email/password + OAuth)
- [ ] Agent/broker profiles with branding (logo, colors, contact info)
- [ ] Saved campaign history — view, edit, and re-export past kits
- [ ] Role-based access (admin/agent/team member)

## State MLS Compliance Expansion
- [ ] Per-account state compliance selector
- [ ] Add compliance configs for all 50 states
- [ ] Only allow users from states with verified compliance rules
- [ ] Compliance rule versioning — track MLS rule changes over time
- [ ] Automated compliance audit log per generated description

## Manual Listing Entry
- [ ] Full manual entry form (price, beds, baths, sqft, features, remarks)
- [ ] Support for old listing numbers/links as input
- [ ] Photo upload with drag-and-drop and reordering
- [ ] Video upload support (walkthrough tours, drone footage)
- [ ] Media library — save and reuse approved photos across campaigns

## Enhanced Ad Generation
- [ ] A/B test variations — generate competing versions with slight tweaks
- [ ] Seasonal/event-aware copy (holiday themes, market conditions)
- [ ] Agent voice training — learn an agent's writing style from past listings
- [ ] Multilingual ad generation (Spanish, Mandarin, etc.)
- [ ] Property type-specific templates (luxury, commercial, land, rental)

## Direct Platform Posting
- [ ] Facebook/Instagram API integration (OAuth + post scheduling)
- [ ] Google Ads API integration (campaign creation)
- [ ] Twitter/X API integration
- [ ] Scheduling — set post date/time per platform
- [ ] Performance tracking — pull back engagement metrics

## Analytics & Insights
- [ ] Dashboard with campaign performance overview
- [ ] Which ad variations get the most engagement
- [ ] Cost-per-lead tracking per platform
- [ ] Market comparison — how your listing stacks up against area comps

## SaaS / White-Label Features
- [ ] Multi-tenant architecture (brokerage → agents hierarchy)
- [ ] White-label branding per brokerage (custom domain, logo, colors)
- [ ] Usage metering and billing (Stripe integration)
- [ ] Subscription tiers (free trial, basic, pro, enterprise)
- [ ] Admin dashboard for brokerage managers

## Advanced Scraping
- [ ] Support additional listing sites (Redfin, Realtor.com, Homes.com)
- [ ] MLS IDX feed integration (direct data, no scraping needed)
- [ ] Auto-detect listing site from URL
- [ ] Scrape neighborhood data (schools, walkability, nearby amenities)

## UX Enhancements
- [ ] Dark mode
- [ ] Campaign templates — save and reuse generation settings
- [ ] Batch generation — process multiple listings at once
- [ ] Real-time preview — see ad copy update as you tweak inputs
- [ ] Mobile-optimized experience for agents in the field
- [ ] Keyboard shortcuts for power users

## Integrations
- [ ] CRM integration (Follow Up Boss, KVCore, Sierra Interactive)
- [ ] Email marketing (Mailchimp, Constant Contact) — export ads as email templates
- [ ] Canva integration — auto-generate social media graphics with listing photos + ad copy
- [ ] Zapier/Make webhooks for automation workflows
